Getting Started

  1. Install

    Stello runs on Node.js, you'll need to install that first if you don't have it already.

    Then simply use the Node.js package manager, npm, to get Stello:

    npm install -g stello

    Voila! If all worked correctly you should now have access to the stello command.

  2. Gather Your Trello Info

    Stello users your public developer key and an application specific token to grab pull data from your Trello boards.

    After authenticating with Trello visit this page to grab your public and private developer keys.

    Then use the form below to get an appliation token for Stello:
